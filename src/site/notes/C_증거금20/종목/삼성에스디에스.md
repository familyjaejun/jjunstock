# 삼성에스디에스

## 요약
- 동사는 1985년 설립, 2014년 상장한 회사로 삼성 관계사의 핵심 IT서비스 파트너로 성장함. 동사는 IT기술 역량을 활용하여 고객의 요구에 맞는 다양한 IT서비스를 제공하고 있음. 당사가 제공하는 IT서비스는 크게 클라우드 서비스, SI, ITO의 3개 분야임. 인도, 베트남, 중국에 글로벌 개발센터와 원격근무방식을 도입하여 디지털 혁신을 선도하고 있음.
